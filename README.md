# RailYatra

RailYatra is a frontend railway web app built with HTML, CSS, and JavaScript.

## Features
- Search trains between stations
- Train schedule lookup
- PNR status mock view
- Dark mode toggle
- Responsive UI
- Deployed on GitHub Pages

## Live Demo
https://jamxgnet07.github.io/train-route-app/

## Tech Stack
- HTML
- CSS
- JavaScript
- GitHub Pages

Frontend version is complete. Real railway API integration is planned in the next phase.
